# Expento 🪙
**Premium Glassmorphism Finance Tracker**

Expento is a high-fidelity, mobile-first personal finance management application designed with a modern **Glassmorphism** aesthetic. It provides a seamless, interactive experience for tracking expenses, managing budgets, and setting reminders.

---

## ✨ Core Features

### 📊 Real-time Dashboard
- **Total Balance Tracking:** Live updates of your current financial standing.
- **Dynamic Stats:** Monitor total income and expenses at a glance.
- **Quick Actions:** High-speed access to adding transactions, scanning QR codes, and sending/requesting payments.

### 💸 Transaction Management
- **Full CRUD:** Add, Edit, and Delete transactions with ease.
- **Category Intelligent:** Automatically maps transactions to specific budget limits.
- **Smart Filtering:** Filter transactions by Month, Week, or Category.
- **Premium Receipts:** View and download deep-design digital receipts for every transaction.

### 📅 Notes & Reminders
- **Vibrant Glass UI:** A dedicated screen for financial notes and reminders with floating background blobs.
- **Task Management:** Group reminders by date and toggle completion status.
- **Integrated Icons:** Categorize your thoughts with expressive emojis and modern typography.

### 📈 Analysis & Insights
- **Visual Breakdown:** Clean, colorful charts showing where your money goes.
- **Category Analytics:** Identify your highest spending areas instantly.

### 🎯 Budgeting & Savings
- **Monthly Limit:** Set a global spending cap for the entire month.
- **Category Budgets:** Set specific targets for Food, Transport, Shopping, etc.
- **Overspend Alerts:** Visual cues (Red Progress Bars) when you cross 90% of your budget.
- **Savings Progress:** Track your balance against a set savings goal with immersive progress animations.

### ⚙️ Premium Settings
- **Real-time Currency Conversion:** Instantly convert the entire app's data between USD, INR, EUR, GBP, JPY, CAD, and AUD.
- **Theme Engine:** Seamlessly toggle between **Light** and **Dark** modes.
- **Account Management:** Update profile details (Name, Email, Username) with real-time UI synchronization.

### 🛡️ High-Fidelity UI/UX
- **Unified Custom Modals:** 100% custom deletion confirmations and success alerts (No generic browser pop-ups).
- **Interactive Navigation:** Floating "pill-style" navigation bar with spring-based animations.
- **Ambient Backgrounds:** Dynamic, blurred background elements that shift with your theme.

---

## 🛠️ Technology Stack
- **Backend:** Python (Flask)
- **Database:** SQLite (SQLAlchemy)
- **Frontend:** Vanilla HTML5, CSS3 (Modern Flex/Grid), JavaScript (ES6+)
- **Icons:** Google Material Symbols (Rounded)
- **Typography:** Inter / Outfit / Roboto

## 🚀 Getting Started

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Application
```bash
python app.py
```

### 3. Access the App
Open your browser and navigate to:
`http://127.0.0.1:5000`

## 🎨 Design Philosophy
Expento is built on the principle of **"Visual Excellence"**. Every component utilizes:
- **Frosted Glass (Glassmorphism):** High blur levels and subtle borders for a premium feel.
- **Micro-Animations:** Smooth transitions for modal entries and tab switching.
- **User-Centric Layout:** Minimalist design with high-contrast elements for accessibility.
